API Reference
=============

This section contains the complete API documentation for all modules in the Expense Predictor.

Modules Overview
----------------

The Expense Predictor consists of the following modules:

* **model_runner** - Main entry point and model training orchestration
* **helpers** - Data preprocessing and utility functions
* **security** - Security validation and sanitization functions
* **config** - Configuration management
